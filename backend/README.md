Laravel + Filament admin backend for the KAV landing template.

What it provides:
- Filament admin at `/admin`
- public API for the frontend showcase
- content blocks, team, catalog items, gallery, reviews, contacts, SEO
- starter seed data for demo installation
- install profiles: `service`, `medical`, `corporate`

Useful commands:
- `php artisan template:install service --fresh --force`
- `php artisan template:install medical --fresh --force`
- `php artisan template:install corporate --fresh --force`
- `php artisan template:seed-demo medical --fresh --force`

Use `php8.2` on Beget.
